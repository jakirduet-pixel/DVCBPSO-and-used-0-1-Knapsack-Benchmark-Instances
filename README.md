# DVCBPSO-and-used-0-1-Knapsack-Benchmark-Instances
C# implementation of DVCBPSO with 0–1 knapsack benchmark instances for experimental evaluation.

This repository provides the C# implementation of the proposed DVCBPSO (Dynamic Velocity-Controlled Binary Particle Swarm Optimization) algorithm, along with the 0–1 knapsack benchmark instances used in the experimental study. The implementation supports reproducibility of published results and enables researchers to perform comparative analysis, parameter tuning, and further extensions in binary optimization research.
